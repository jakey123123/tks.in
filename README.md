# 3D Print Shop

A full-stack web application for displaying 3D printed items, managing orders.

## Project Structure

- `frontend/` - React website for displaying items and orders
- `backend/` - Node.js/Express API server
- `docs/` - Documentation

## Features

- Display 3D printed items with images and descriptions
- Add new items to inventory
- Order management system
- Live order notifications on dashboard

## Quick Start

### Backend
```bash
cd backend
npm install
npm start
```
Server runs on `http://localhost:5000`

### Frontend
```bash
cd frontend
npm install
npm start
```
App opens at `http://localhost:3000`

See `docs/SETUP.md` for detailed setup instructions.
